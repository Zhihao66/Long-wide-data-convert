🎯 核心功能演示 (Core Features)

【功能 1】Python API - 宽转长
   from data_transformer import DataTransformer
   import pandas as pd
   
   df = pd.DataFrame({'id': [1, 2], 'Math': [85, 90], 'English': [92, 88]})
   transformer = DataTransformer()
   result = transformer.wide_to_long(df, id_vars='id')

【功能 2】Python API - 长转宽
   result = transformer.long_to_wide(df, index='id', columns='subject', values='score')

【功能 3】命令行工具
   data-transformer wide-to-long input.csv output.csv --id-vars id,name

【功能 4】文件支持
   - CSV 格式 (逗号分隔)
   - Excel 格式 (.xlsx, .xls)
   - JSON 格式

🎓 理解宽数据和长数据 (Understanding Formats)

宽格式 (Wide Format):
  | ID | 2020 | 2021 | 2022 |
  |----|------|------|------|
  | 1  | 100  | 120  | 150  |

长格式 (Long Format):
  | ID | Year | Value |
  |----|------|-------|
  | 1  | 2020 | 100   |
  | 1  | 2021 | 120   |
  | 1  | 2022 | 150   |

用途：
  - 宽格式 → 数据展示、比较分析
  - 长格式 → 统计建模、时间序列分析
