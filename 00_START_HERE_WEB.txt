【第 2 步】启用 GitHub Pages (1分钟)

    1. 打开：https://github.com/你的用户名/data-transformer

    3. 左边找 Pages
    4. Branch 选择 "main"
    5. Folder 选择 "/" 或 "/web"
    6. 点击 Save


【第 3 步】访问你的网站 (立即)
