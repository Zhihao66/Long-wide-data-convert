==================== 🚀 部署步骤 ====================

【步骤1】推送到 GitHub (2 分钟)

   cd e:\数据转换
   git add .
   git commit -m "Add web version with GitHub Pages support"
   git remote add origin https://github.com/你的用户名/data-transformer.git
   git push -u origin main

【步骤2】启用 GitHub Pages (1 分钟)

   1. 打开 https://github.com/你的用户名/data-transformer
   2. 点击 Settings (设置)
   3. 左侧找 Pages
   4. Branch 选择: main
   5. Folder 选择: / (根目录) 或 /web
   6. 点击 Save

【步骤3】访问你的网站 (几分钟后)

   https://你的用户名.github.io/data-transformer/

==================== 🆘 常见问题 ====================

Q: 网站显示 404 错误？
A: 1. 确认 Settings → Pages 已启用
   2. 检查分支是否为 main
   3. 检查选择的文件夹是 / 或 /web
   4. 等待几分钟让 GitHub 部署

Q: 网站显示但样式不正确？
A: 1. 清除浏览器缓存 (Ctrl + Shift + Del)
   2. 使用隐私窗口测试
   3. 检查 HTML 中的路径

Q: JavaScript 错误？
A: 1. 打开浏览器 F12
   2. 查看 Console 标签
   3. 检查错误信息

Q: 如何使用自己的域名？
A: 1. Settings → Pages → Custom Domain
   2. 输入你的域名
   3. 在域名提供商设置 DNS
