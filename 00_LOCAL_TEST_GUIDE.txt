## 🚀 立即测试（3个简单步骤）

### 步骤 1️⃣：打开网站
- 在浏览器中访问：**http://localhost:8000**
- 你应该看到一个漂亮的紫色界面

### 步骤 2️⃣：使用测试数据
在"输入数据"框中粘贴以下数据：

```
id,name,Math,English,Science
1,Alice,85,92,88
2,Bob,90,88,92
```

### 步骤 3️⃣：配置转换参数
1. 选择"宽 → 长"模式
2. ID 列填：`id,name`
3. 变量列名填：`subject`
4. 值列名填：`score`
5. 点击"🔄 开始转换"

### 结果 ✨
你应该看到转换后的长格式数据：
```
id,name,subject,score
1,Alice,Math,85
1,Alice,English,92
1,Alice,Science,88
2,Bob,Math,90
2,Bob,English,88
2,Bob,Science,92
```

## 📝 测试场景

### 场景 1：学生成绩（宽→长）

**输入**：
```
StudentID,Name,Math,Science,English
101,张三,85,92,88
102,李四,90,88,95
```

**参数**：
- ID 列：`StudentID,Name`
- 变量列名：`Subject`
- 值列名：`Score`

**输出**：6 行长格式数据

---

### 场景 2：销售数据（宽→长）

**输入**：
```
Region,2020,2021,2022
East,100000,120000,140000
West,80000,95000,120000
```

**参数**：
- ID 列：`Region`
- 变量列名：`Year`
- 值列名：`Sales`

**输出**：6 行长格式数据

---

### 场景 3：长→宽逆向转换

使用上面的长格式输出，转换回宽格式

**参数**：
- 索引列：`Region`
- 列列：`Year`
- 值列：`Sales`

**输出**：应该是原始的宽格式

## 📱 响应式测试（手机显示）

按照以下步骤测试手机适配：

1. **打开开发者工具**：按 `F12`
2. **切换到手机模式**：按 `Ctrl + Shift + M`
3. **选择设备**：
   - iPhone 12 (390x844)
   - Galaxy S20 (360x800)
   - iPad (768x1024)

4. **检查**：
   - ✓ 界面是否单列显示
   - ✓ 按钮是否可点击
   - ✓ 文本是否可读
   - ✓ 输入框是否合适大小

## 🎨 快速自定义测试

### 修改标题
编辑 `web/index.html` 第 51 行：
```html
<h1>📊 你的标题</h1>
```
刷新页面后，标题会立即更新！

### 修改颜色
编辑 `web/index.html` 第 16 行：
```css
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
/* 改为你喜欢的颜色代码 */
```
保存后刷新，背景色会改变！
